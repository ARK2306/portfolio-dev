# aryanreddy.dev

Personal portfolio site for Aryan Reddy Kalluri — Backend & Cloud Engineer.

Built with [Astro](https://astro.build/) and [Tailwind CSS v4](https://tailwindcss.com/). Deployed to GitHub Pages via GitHub Actions.

## Local development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` automatically deploy to [ark2306.github.io](https://ark2306.github.io) via the GitHub Actions workflow in `.github/workflows/deploy.yml`.

To enable: go to **Settings → Pages → Source** and set it to **GitHub Actions**.

## Customization

All content lives in `src/config.ts`.
